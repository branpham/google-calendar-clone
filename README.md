# Google Calendar Clone

A complete 1:1 clone of Google Calendar built with React, TypeScript, and Tailwind CSS.

## Features

- ✅ **Multiple Views**: Month, Week, Day, Year, and Schedule views
- ✅ **Event Management**: Create, edit, delete, and manage events
- ✅ **Recurring Events**: Support for daily, weekly, monthly, and yearly recurrence
- ✅ **Multiple Calendars**: Create and manage multiple calendars
- ✅ **Dark Theme**: Matches Google Calendar's dark mode aesthetic
- ✅ **Responsive Design**: Works on desktop and mobile
- ✅ **Local Storage**: Events saved automatically to localStorage

## Quick Start

### Installation
```bash
git clone https://github.com/branpham/google-calendar-clone.git
cd google-calendar-clone
npm install
npm run dev
```

The app will open at `http://localhost:3000`

### Build for Production
```bash
npm run build
npm run preview
```

## Technologies

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Zustand** - State management
- **date-fns** - Date manipulation
- **Tailwind CSS** - Styling
- **Vite** - Build tool
